超级终端

### 把aria2c文件放到/sdcard目录
### 建立在data/data内aria2目录  
 
```c  
mkdir -p /data/data/aria2  
```  
### 复制aria2c文件到aria2目录  

```c  
cp /sdcard/aria2c /data/data/aria2/  
```  


### 赋予744管理权限文件  

```c  
chmod 744 /data/data/aria2/aria2c  
```  
  

### 启动aria2的RPC模式  

```c    
./aria2c --enable-rpc --rpc-listen-all  
```  



### 查看aria2版本  

```c    
./aria2c -v  
```
